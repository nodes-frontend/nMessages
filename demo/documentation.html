Global configuration

Some options are only configurable on a global level:

Option Type Details
horizontalPosition string Supported values:
  • 'left'
  • 'center'
  • 'right'
verticalPosition string Supported values:
  • 'top'
  • 'center'
  • 'bottom'
maxNumber int Limit the maximum number of messages displayed at one time

Make sure the nMessagesProvider is injected in to your applications config module, and use the providers configure method:


angular.module('application').config(function(nMessagesProvider) {
nMessagesProvider.configure({
verticalPosition: 'top',
horizontalPosition: 'right',
maxNumber: 3
});
});

Basic usage:

To get started, inject the nMessages provider into your controller, and call the create method: 


angular.module('application').controller('appCtrl', function(nMessages) {
nMessages.create('The most basic message type available');
});

Advanced usage:

You can configure the individual messages when calling the create method: 


vm.message = {
type: 'success',
content: 'Such message',
dismissButton: true,
dismissButtonHtml: '×',
dismissOnClick: true,
dismissOnTimeout: false,
timeout: 4000
};

Options:

Option Type Details
type string Use this to add a contextual class to your message (ie. 'success', 'error' etc.)
content string or html You can pass both a string or html, giving you the freedom to customize the messages as you please
dismissButton boolean Shows or hides the dismiss button
dismissButtomHtml string or html You can pass both a string or html, giving you the freedom to customize the close button (ie. use an icon font)
dissmissOnClick boolean If boolean is true, the message can be closed by pressing anywhere on the message - not just the close button
dismissOnTimeout boolean Should the message auto-close?
timeout int (ms) If boolean is true, the message will be auto-closed after this duration
